  • Fridgly

    Fridgly is a kitchen ingredient management and recipe seach app. It allows the user to manage ingredients in the 'fridge' and 'pantry', and from these collections, select ingredients to search for recipes.

    Fridgly uses the Yummly API to send recipes back to the user. The back-end is built on Node.js, with a PostgreSQL database. The front-end is built with Backbone.js.

  • Ragnar

    Ragnar is a sidescrolling hack and slash game where that puts the player in control of Ragner the dwarf. Ragnar must travel across the jungle, battle foes and kill the Boss at the end of the level to win the game.

    Ragnar was built using the Phaser JavaScript library to give the game physics. The back-end is Node.js, with a PostgreSQL database which saves the player's high scores.

  • GameSage

    GameSage is a video game search app. It allows the user to log in, search for games, view a game's specific details(screenshots, game reviews, list of characters, synopsis). You can also save games to your favorites and rate those games.

    GameSage was built entirely on Rails. It utilizes the GiantBomb API for the search results.
